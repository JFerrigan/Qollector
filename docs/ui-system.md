# VinylCollector UI System

## Visual Direction
- Flat, modern, pastel-first palette.
- Large soft surfaces with crisp typography and restrained contrast.
- Minimal chrome, rounded surfaces, and lightweight line accents instead of heavy shadows.
- Browsing should feel like moving through a personal collection shelf rather than a spreadsheet.

## Tokens
- Background presets: warm paper, rose tint, mint tint, powder blue, peach accent.
- Font presets: 5 curated options chosen for readability and personality, applied through shared theme tokens rather than per-view overrides.
- Typography roles: display, section title, card title, metadata, caption.
- Spacing scale: 4, 8, 12, 16, 20, 28, 36.
- Radii: 12, 20, 28.
- Borders: thin, low-contrast pastel lines.

## Component Rules
- Every page uses a consistent header with page title and short supporting copy when useful.
- Record cards pair icon artwork with text metadata and compact tags.
- Inputs and settings rows use the same rounded surface treatment as cards.
- Empty states are calm and illustrative, not loud.
- Motion should be limited to page swipes, sheet presentation, and icon preview updates.

## Interaction Rules
- Library prioritizes scanning and recognition.
- Add/Edit prioritizes low-friction form entry.
- Search results should mirror the library card language.
- Settings should explain rating mode clearly because it affects global presentation.
- Settings should present font and background options as fixed presets, with immediate whole-app preview when changed.
